📋 Étape 2 : Générez les fichiers CDE et LOT

Ces fichiers sont nécessaires pour créer la commande et les lots dans Odoo

ℹ️ Que faire ensuite ?

  • 1. Téléchargez les fichiers CDE et LOT ci-dessous
  • 2. Importez-les dans Odoo dans cet ordre : CDE puis LOT
  • 3. Odoo va créer une réception et générer un ID
  • 4. Récupérez cet ID de réception dans Odoo
  • 5. Passez à l'étape 3 pour générer le fichier de réception
